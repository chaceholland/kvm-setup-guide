üîë Key Principles:
  • EDID in Every Chain: All 6 connections have inline EDID emulators to prevent window rearrangement
  • EDID Requires HDMI: All signals convert to HDMI before EDID, then convert to final input type after EDID
  • Unique Input Types: Each computer uses a different input type on both monitors for simultaneous DDC/CI switching
  • Better Display Control: Cmd+1/2/3 switches both monitors simultaneously to HDMI/DP/USB-C
